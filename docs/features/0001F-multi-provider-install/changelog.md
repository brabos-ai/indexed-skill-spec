# Changelog: 0001F
> **Date:** 2026-03-10 | **Branch:** feature/0001F-multi-provider-install

## Summary

Expands the idx-skill CLI from 4 to 18 AI providers with correct destination paths, adds non-interactive `--provider` flag, `check` command for provider detection, and auto-selection of existing provider folders. Includes path fixes for codex and antigravity providers.
